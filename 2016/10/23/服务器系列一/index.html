服务器系列一

By Xiangbing Huang

发表于 2016-10-23

文章目录

下午在阿里云服务器上安装mongodb，弄了好长时间，踩了好多坑，尼玛、、、

首先连接到阿里云服务器，操作系统Windows Server 2012 R2

进入mongodb官网https://www.mongodb.com/，死活下不下来.msi的，可能是服务器的安全级别比较高，网上好像说要下.zip的，但我感觉跟之前的版本不一样，怎么都下不下来。我就想安装ftp，先下到本地，然后传上去，百度了半天ftp还是安装失败，沮丧、、这个时候我就想到能不能下载个qq从外面传进去，尼玛没想到qq也不能用还弄成了保护模式，还要申诉找回密码。

最后想到硬盘共享，又找到一篇文章http://zhidao.baidu.com/link?url=hvVKJNOQU5owD1eN7gmMgAt4HK479BGCROt61KTiLk55_JjA07Qb_lqt6O-6lYtEjgfW87hkyxdGH1UT2FmhfpOwsk5jGCoIjG-9n1nH96i于是把mongodb-win32-x86_64-2008plus-ssl-3.2.10-signed.msi下载到d盘，本地资源勾选了d盘，远程连接之后（用户名Administrator 密码是你服务器开机密码），我把它拷贝到了c盘，反正服务器就一个盘。

接下来就参考了这篇文章：http://blog.csdn.net/qq_16313365/article/details/52241357 还有系列（三）一步一步来就可以啦。

注意：目录有空格加双引号
将mongodb的bin目录添加到Path中。

最后在cmd中输入mongo，显示connecting to: test
